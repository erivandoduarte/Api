# Api
Projeto Api  Faculdade
# Dados Técnicos

- Linguagem: C#

- Ferramenta de desenvolvimento: Microsoft Visual Studio Professional 2017
 
- Framework: .NET Framework 4.5

- ORM: Entity Framework 6.1

- Dependency Injection 
- SimpleInjector

- Sql Server 
#Instalação

1. Faça download do projeto.

2. Descompacte o arquivo .zip.

3. Abra o projeto

4. Abre "Package Manager Console" e execulte os seguintes comandos:
  
4.1 - Add-Migration initial,
  
4.2 - Update-Database 

5. Execute-o (F5)
